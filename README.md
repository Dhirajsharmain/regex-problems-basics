**UC3-ValidatingEmail**

_Description :_ As a User need to enter a valid email
- E.g. abc.xyz@bl.co.in - Email has 3 mandatory parts (abc, bl
  & co) and 2 optional (xyz & in) with
  precise @ and . positions.
